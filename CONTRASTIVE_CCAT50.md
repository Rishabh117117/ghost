# Contrastive author-isolation on CCAT50 — two-arm experiment

**Question:** the generic diagnostic (GENERIC_TEST.md) showed the voice ghost leaks (0.32 of
its own-domain win transferred to unrelated wikitext). Hypothesis: shortcut learning — plain
next-token CE buys generic prose fluency, not author style. Test on a public, controlled
corpus with a same-register control, and test a contrastive hinge as the cure.

**Verdict: leak REPLICATED (0.471) | cure FAIL (helped, but missed the bar).**

## Setup
- **Corpus:** CCAT50 / Reuter_50_50 (UCI id 217), 50 authors × 50 train + 50 test docs, all
  CCAT register by design. Data stays local under `data/ccat50/` (gitignored); reproduced by
  `download_ccat50()` in `contrastive_ccat50.py`.
  *Deviation:* `ucimlrepo` cannot serve id=217 (text dataset, not importable via the API) —
  used the spec's fallback, the UCI static zip.
- **Target author:** AaronPressman (alphabetically first). **Contrast authors (10):**
  AlanCrosby, AlexanderSmith, BenjaminKangLim, BernardHickey, BradDorfman, DarrenSchuettler,
  DavidLawder, EdnaFernandes, EricAuchard, FumikoFujisaki — 20 C50test docs each (n=200).
- **Third column:** wikitext-2-raw-v1[test] (n=80), same loader/filter as `generic_test.py`.
- Base `Qwen/Qwen3-4B` frozen (bf16, CUDA), seed 0, architecture untouched (imported from
  `ghost.py`). Masking byte-identical to Stage 1/2: doc as assistant turn after the fixed
  neutral prompt; assistant tokens scored, −100 elsewhere.
- *Deviation (comparability):* docs pre-truncated to 220 tokens at the text level so the
  chat-formatted example fits MAX_LEN=256 and **every grid row scores the identical doc
  tokens** (the style-prompt row uses a longer context; scaffold is masked either way).
  Verified: identical scored span under neutral and style prompts.
- **Worked masking example (described, not quoted):** doc 0 → 238 tokens total, 220 scored
  (exactly the truncated doc, the assistant span), 18 masked scaffold (user prompt + headers +
  empty think block + trailing im_end).

## Phase 0b — neutral paraphrases (the negatives)
One paraphrase per target train doc, generated by the frozen base itself (chat template,
thinking disabled, temp 0.3, top_p 0.9, retry once at 0.7; keep if non-empty and token-ratio
in [0.5, 2.0]). **Survival: 50/50 kept, 0 dropped** (no retries exhausted). Cached at
`data/ccat50/pairs_target.jsonl` (gitignored).

## Arms
- **Arm 1 — plain** (`ghost_ccat_plain.pt`): exact v03 recipe (D_GHOST 224, lr 1e-4, wd 0.01,
  patience 3, seeded 85/15 split = 42/8). Early-stopped epoch 13, best val CE 3.028 (epoch 10).
- **Arm 2 — contrastive** (`ghost_ccat_contrastive.pt`): same data/knobs; per doc i,
  `L_i = CE_g(doc_i) + 1.0 * max(0, CE_b(n_i) − CE_g(n_i))` with `CE_b(n_i)` precomputed once
  on the frozen base. Early-stop on the val value of the same combined objective.
  Early-stopped epoch 14, best val 3.280 (epoch 11).
- Both arms: resumable per-epoch checkpointing; keep-awake power assertion (Modern-Standby
  laptop). Note: 50 docs is a smaller corpus than Stage 1's (~745 turns); both arms share it,
  so the comparison is internal and fair.

## Verdict

1. **Leak REPLICATED — decisively.** Plain leak fraction **0.471** (threshold ≥0.15): nearly
   half the "author win" transfers to *same-register, different-author* text, and +15.5%
   even to wikitext. On this controlled corpus the smear is *worse* than the 0.32 measured
   on personal data — strong support for the shortcut-learning hypothesis: plain CE buys
   register/prose fluency more than author identity.
2. **Cure FAIL — directionally right, insufficient.** The hinge kept 86% of own-win
   (0.86 ≥ 0.6 ✓) and cut the leak fraction 0.471 → 0.332 (ratio 0.71, needed ≤0.5 ✗; abs
   0.332, needed ≤0.10 ✗). It moves the right direction but a λ=1.0 hinge against
   *self-paraphrases of the same docs* doesn't reach author isolation. Likely because the
   negatives are too narrow: paraphrases of the target's own articles share the author's
   topics, so "don't help the neutral version" still leaves the register/topic shortcut
   open against *other authors'* text.
3. **Style-prompt baseline:** *hurts* on CCAT (−24.8% on target, −25.1% on contrast) — few-shot
   author samples in the prompt make Reuters text *less* predictable for the base. Both ghosts
   beat it easily on column A, so the honesty check passes (no flag).

## What's next (not done here)
- Stronger negatives: contrast-author docs (not just self-paraphrases) as the hinge set —
  directly penalize improvement on other CCAT authors.
- λ sweep / larger λ (explicitly out of scope for this task's single-knob run).
- The bank implication stands: a plain-CE "style" ghost is roughly half generic-register
  corrector — routing decisions based on own-domain ppl wins will overestimate specialization.
