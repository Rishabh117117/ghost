## Gotchas learned this session
- Sandbox pip: use `pypi.org` (allowed); `download.pytorch.org` was blocked.
  CPU venv used plain `pip install torch transformers==5.9.0 ...`.
- `transformers` 5.9.0 + torch 2.x: `from_pretrained(..., dtype="auto")` with
  `TypeError` fallback to `torch_dtype` is already handled in the code.
- Pod image `runpod/pytorch:2.4.0-py3.11-cuda12.4.1-devel-ubuntu22.04`
  (in `runpod_launch.py`); adjust if RunPod has retired the tag.
- CCAT50 comes from the UCI static zip (in `download_ccat50()`), not HF ‚Äî
  the pod needs plain outbound HTTPS to archive.ics.uci.edu.
- The smoke run's `results/smoke/` is committed as plumbing proof; the real
  run writes `results/sweep_ccat50/` + root `SWEEP_CCAT50.md`/`results.json`.
